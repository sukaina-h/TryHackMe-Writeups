# Networking Fundamentals

## Room Overview
This room covers core networking concepts required to understand how systems communicate and how attacks travel across networks.

## Topics Covered
- TCP/IP model
- IP addressing
- Ports and protocols
- DNS, HTTP, HTTPS
- LAN vs WAN

## Key Concepts Learned
- How data moves across networks
- Importance of ports and services
- Relationship between networking and security attacks

## Security Relevance
- Network knowledge is required for traffic analysis
- Many attacks exploit open ports and misconfigured services
- Forms the foundation for tools like Nmap and Wireshark

## Practical Takeaways
- Ability to identify common protocols
- Improved understanding of attack surfaces
- Prepared for network scanning and monitoring labs
