# Command Line Fundamentals

## Room Overview
This room builds confidence using the command line, a critical skill for cybersecurity analysis and automation.

## Commands Practiced
- grep
- find
- echo
- pipe (|)
- redirection (>, >>)
- history

## Topics Covered
- Searching files and directories
- Filtering command output
- Command chaining
- Input/output redirection

## Practical Use Cases
- Analyzing log files
- Searching for indicators of compromise
- Automating repetitive tasks

## Key Takeaways
- Command-line efficiency improves investigation speed
- Logs and system data are often analyzed via CLI
- Essential skill for SOC analysts and penetration testers
