# Windows & Active Directory Fundamentals

## Room Overview
This room introduces Windows operating systems and Active Directory concepts commonly used in enterprise environments.

## Topics Covered
- Windows OS architecture
- User accounts and groups
- Active Directory basics
- Domain controllers
- Authentication and authorization

## Tools & Concepts
- Windows Event Logs
- Group Policy
- User and computer objects
- Domain vs local accounts

## Practical Exercises
- Identified Windows user and group roles
- Explored Active Directory components
- Learned how authentication works in a domain environment

## Security Relevance
- Active Directory is a frequent attack target
- Misconfigured permissions can lead to privilege escalation
- Understanding AD is critical for both attackers and defenders
