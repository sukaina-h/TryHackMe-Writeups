# Introduction to Cybersecurity

## Room Overview
This room introduces the fundamentals of cybersecurity, including what cybersecurity is, why it matters, and the different domains within the field.

## Topics Covered
- What is cybersecurity
- CIA Triad (Confidentiality, Integrity, Availability)
- Threats vs vulnerabilities
- Blue Team vs Red Team
- Real-world cyber attacks

## Key Concepts Learned
- Importance of protecting data and systems
- How cyber attacks impact individuals and organizations
- Overview of defensive and offensive security roles

## Practical Takeaways
- Developed a foundational security mindset
- Understood the role of cybersecurity in modern digital systems
- Built awareness of common cyber threats

## Relevance to Security Roles
This room provides the baseline knowledge required for all cybersecurity roles, including SOC Analyst, Security Engineer, and Penetration Tester.
