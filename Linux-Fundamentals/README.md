# Linux Fundamentals

## Room Overview
This room focuses on basic Linux operating system usage and command-line navigation, which is essential for cybersecurity professionals.

## Commands & Tools Used
- ls
- cd
- pwd
- cat
- nano / vi
- chmod
- chown
- sudo

## Topics Covered
- Linux directory structure
- File permissions and ownership
- User and group management
- Basic system commands

## Practical Exercises
- Navigated the Linux file system
- Viewed and edited files
- Modified permissions using chmod
- Executed commands with elevated privileges

## Key Takeaways
- Linux is widely used in servers and security tools
- Understanding permissions is critical for privilege escalation prevention
- Command-line skills are essential for incident response and pentesting
